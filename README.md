# Jogo-OContoDasFolhas
Jogo criado para projeto durante o curso de C#, do programa Entra21 em 2025. Com base em "cozy games", esse jogo teve inspiração em personagens fictícios, os Barbafolhas são criaturas mágicas que vivem em harmonia no seu mundo, criando receitas, forjando armas e curando a árvore protetora. Os Ruklins são os vilões que aparecem para batalhar.
